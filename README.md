# Student Performance Analysis

## Project Overview
This project analyzes student performance based on study hours and subject scores using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Google Colab

## Key Insights
- Students who studied more hours scored higher in math.
- Science had slightly higher average scores than math.
- Strong positive correlation between study time and performance.

## Purpose
This is a beginner data analytics project created as part of my preparation
